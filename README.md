# cryptoLiveWeb
cryptocurrency live chart providing simple services<br>

Stack<br>
- RDS(MariaDB)<br>
- DynamoDB
- Node.js
- React.js
- Socket.io
<br><br>

Live Chart refreshed every 60 seconds<br>
Latest News Article refreshed every 1 hour<br>

API Source: www.cryptocompare.com<br>
